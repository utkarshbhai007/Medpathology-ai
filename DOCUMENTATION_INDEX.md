## 🎓 Learning Path

### Day 1: Setup & Overview
1. Read **README.md** (15 min)
2. Follow **SETUP.md** (30 min)
3. Check **RUNNING_SERVICES.md** (5 min)
4. Print **QUICK_REFERENCE.md** (keep handy)

### Day 2: Learn the System
1. Read **USER_GUIDE.md** - User Roles section (20 min)
2. Study **FLOWCHARTS.md** (15 min)
3. Login with demo credentials
4. Explore each dashboard (30 min)

### Day 3: Practice Workflows
1. Follow Lab Admin workflow in **USER_GUIDE.md**
2. Upload a test report
3. Follow Doctor workflow
4. Review the report
5. Follow Patient workflow
6. View health portal

### Day 4: Master Features
1. Read Feature Guide in **USER_GUIDE.md**
2. Try each advanced feature
3. Use **QUICK_REFERENCE.md** for quick lookups
4. Experiment with different scenarios
